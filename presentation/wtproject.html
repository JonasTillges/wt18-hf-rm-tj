  • User Stories

Frontend / Anforderungen

Unser Anspruch an das Frontend:
  • Mobile First
  • Clean Design

Backend / Buildsystem

Gulp hilft uns bei:
  • automatisiert Prozesse
  • SASS/SCSS Kompelierung
  • Zusamenführung von Javascript Dateien
  • Minifizierung von Javascript/CSS

Backend / Datenbank und SSL

Docker + MongoDB / Lets Encrypt
  • Docker Container für MongoDB
  • LetsEncrypt zur SSL Zertifikaterzeugung

Application Framework und Server

Vue.js und Express.js
  • Vue.js als Schnittstelle zwischen Client & Server
  • Express.js als Request Handler

Appeareance und Styling

Bootstrap 4 und Sass
  • Bootstrap 4 als Frontendframework
  • SASS als Extension Language für CSS
